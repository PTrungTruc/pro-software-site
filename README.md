1) Hướng dẫn giản đơn để chạy trên localhost:
+ Với folder api: Chạy câu lệnh sau "npm run nodemon --prefix api" (Thế là đã chạy được backend rồi).
+ Với folder dashboard: Chạy câu lệnh "npm run dev --prefix dashboard" (Thế là chạy được frontend rồi).
+ Lưu ý: các câu lệnh chạy từ folder gốc chứa 2 folder trên

2) Cài đặt:
- Với yêu cầu cần đảm bảo đã cài Nodejs để chạy được.
- Kiểm tra các thiết lập trên file .env; điều chỉnh đường dẫn trên file server.js.
